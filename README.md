# Wisely Web Test

## Requirements

* [Docker Desktop]()
* Node 16+

## Getting Started

- run `docker-compose up`

### JetBrains IDEs
If using a JetBrains IDE you may need ot install the dependencies for `api`
and `frontend` locally _before_ creating the Docker images. In that case run
the following from the repo root:

```shell
cd api
yarn install
cd ../frontend
yarn install
```
